  synchronizeTimeInterval

Time interval for synchronization of campaign analytics in seconds. Specify this value before [LSCampaignAnalyticsManagerDelegate startGatheringCampaignAnalytics] calling If provided value is less than 300 or bigger than 21600, default value will be used. If no value is provided, default value will be used. Default value is 3600.

@property (nonatomic, assign) NSTimeInterval synchronizeTimeInterval

– synchronizeWithServer

Performs the sending of collected campaign events to server 1000 events is a maximum value of campaign events. If more events is gathered after the last sync - only 1000 last events will be sended. 48 h is a maximum time interval for storing events. Only events, that were collected during last 48 h will be sended.
